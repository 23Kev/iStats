This tool was created by Brian Fiete. The site generation code is hosted on GitHub.

The presentation is very vanilla - please feel free to submit improvements/suggestions through GitHub issues, PRs, or email (bfiete@gmail.com), even in the form of modified HTML files.

TOP AVERAGE/FASTEST LAP: The post-race average/fastest lap time for every entry is sorted, and then the 90th to 95th percentile (fast but non-alien) is averaged. The "+" value shows how much slower this time is from the fastest car time (+0.000 is the fastest car). The "k" value is the average IR from the entries used in the calculation.

Note that race times are listed in your local time.
